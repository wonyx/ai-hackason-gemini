# gemini-server

## setup
```
npm install
```
## development
```
npm run dev
``` 
## build
```
npm run build
```

## deploy
```
gcloud run deploy --source .
```
refer: https://cloud.google.com/run/docs/quickstarts/build-and-deploy/deploy-nodejs-service?hl=ja

### 環境変数
| key | value |
| --- | --- |
| GEMINI_API_KEY | APIキー<br> refer: https://ai.google.dev/gemini-api/docs/api-key?hl=ja |
